#Finance Tracker V2

A full-stack finance management application built with the MERN stack that helps users securely manage their finances, track transactions, and monitor spending habits.

---

## Features

### Authentication & Security
- User Registration
- Secure Login System
- JWT Authentication
- Access & Refresh Tokens
- Protected Routes
- Password Reset via Email
- Email Verification
- Password Encryption using bcrypt
- HTTP-only Cookie Authentication

### User Experience
- Modern React Frontend
- Responsive Design
- Tailwind CSS Styling
- Loading States
- Error Handling
- Form Validation

### Finance Management *(In Progress)*
- Income Tracking
- Expense Tracking
- Transaction Categories
- Financial Dashboard
- Monthly Summaries
- Spending Analytics

---

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Tambeezy1890/Finance-Tracker-V2.git
```

### Install Dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd server
npm install
```

### Run Development Servers

Frontend

```bash
npm run dev
```

Backend

```bash
npm start
```
